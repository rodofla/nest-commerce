# Nest-Commerce API

1. Clonar Proyecto
2. `pnpm install`
3. Clonar el archivo **_.env.template_** y renombrar a **_.env_**
4. Cambiar las variables de entorno

5. Levantar la Base de Datos

```bash
docker-compose up -d
```

6. Ejecutar seed

```bash
pnpm seed:run
```

7. Levantar: `pnpm run start:dev`
